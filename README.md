# Right Side

Right Side is a small local-only iPhone app for drawing a user-authored prompt: a song to hear, a poem to read, an exercise to do, or any other small action that helps the day begin or reset well.

The v1 spec lives in [SPEC.md](SPEC.md).

For setup and device installation instructions, see [RUNNING_ON_IPHONE.md](RUNNING_ON_IPHONE.md).

## Development

- Open `RightSide.xcodeproj` in Xcode.
- Select the `RightSide` scheme.
- For device builds, select an Apple Developer Team in the target signing settings.
- Run on an iPhone Simulator or a connected iPhone.

The project intentionally does not commit developer-team-specific signing configuration.
