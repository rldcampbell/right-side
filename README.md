# Right Side

Right Side is a small local-only iPhone app for drawing a user-authored prompt: a song to hear, a poem to read, an exercise to do, or any other small action that helps the day begin or reset well.

The v1 spec lives in [SPEC.md](SPEC.md).

For personal Mac and iPhone setup, see [RUNNING_ON_IPHONE.md](RUNNING_ON_IPHONE.md).

## Development

- Open `RightSide.xcodeproj` in Xcode.
- Select the `RightSide` scheme.
- In the target signing settings, choose your personal Apple Developer Team.
- Run on an iPhone or iPhone simulator.

The project intentionally does not commit personal signing configuration.
